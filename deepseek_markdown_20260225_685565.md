# StreamHub 2026 — сайт для стримеров и YouTube

Современный адаптивный одностраничный сайт, посвящённый стримерам и YouTube-контенту.  
Сделан с использованием HTML и CSS (без фреймворков). Идеально подходит для размещения на GitHub Pages.

## Особенности

- **Актуальность 2026 года** — все данные (стримеры, игры, расписание) соответствуют февралю 2026.
- **Карточки популярных стримеров** с аватарками, платформами (Twitch, YouTube, Kick) и статистикой.
- **Блок последних YouTube-видео** в виде карточек с превью.
- **Расписание ближайших стримов** с указанием времени, платформы и игры.
- **Топ стримеров по просмотрам** (лидерборд).
- Полная адаптивность под мобильные устройства и планшеты.
- Тёмная тема с неоново-красными акцентами (фирменный цвет стримерской культуры).
- Плавные анимации при наведении.

## Использованные технологии

- HTML5
- CSS3 (Flexbox, Grid, медиа-запросы)
- Google Fonts (Inter)
- Font Awesome 6 (иконки соцсетей и платформ)
- Unsplash (бесплатные стоковые изображения для аватарок и превью)

## Как развернуть на GitHub Pages

1. Создайте новый репозиторий на GitHub (например, `streamhub-2026`).
2. Добавьте в него два файла:
   - `index.html` (скопируйте код выше)
   - `README.md` (этот текст)
3. В настройках репозитория (**Settings → Pages**) выберите ветку `main` и нажмите **Save**.
4. Через пару минут сайт станет доступен по адресу:  
   `https://ваш-username.github.io/streamhub-2026`

## Локальный запуск

Просто откройте `index.html` в любом современном браузере.
